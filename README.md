## Development Environment
- **빠른** 데이터 시각화 웹앱 프레임워크 Streamlit을 활용
- python3.11
- streamlit
- mariadb

## Deployment
```bash
$ pip install pipenv
$ pipenv shell --python 3.11
$ pipenv install
$ streamlit run app.py
```
